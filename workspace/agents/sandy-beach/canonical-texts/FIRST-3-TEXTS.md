# Canonical Texts 1-3 — Reach Out Agent

These are the first 3 messages Sandy Beach sends to every new lead. They are doctrine-level fixed. No AI improvisation. No variation. These texts are the product of years of testing and represent the optimal opening sequence for belief transfer.

## Text 1: Legitimacy + Memory Scratch

**Purpose:** Establish that this is a real person texting, not a bot. Create a "memory scratch" — the lead wonders if they should remember this person.

**Template:**
```
hey {first_name} its sandy from pipeline lite — we connected a while back and i wanted to circle back with you on something
```

**Rules:**
- All lowercase
- No punctuation at end
- "its" not "it's" (intentional — human texture)
- "i" not "I" (intentional — casual, not corporate)
- Must include first name
- Must reference a prior connection (even if vague)
- No CTA. No link. No pitch. Just a door crack.

---

## Text 2: Task Declaration + Human Urgency

**Purpose:** State what you want WITHOUT explaining why. Create natural curiosity gap. The lead should think "what is this about?" not "oh, a sales pitch."

**Template:**
```
was hoping to catch you for a quick min — got something i think youd actually want to see before the week wraps up
```

**Rules:**
- All lowercase
- "youd" not "you'd" (intentional)
- "min" not "minute" (casual compression)
- Time pressure is NATURAL ("before the week wraps up"), not manufactured
- No product name. No company name. No link. No explanation.
- This text ONLY sends if Text 1 got no reply within 24 hours

---

## Text 3: Execution Without Narration

**Purpose:** If still no reply, make the final attempt with maximum human texture. Show you're busy too. Create peer-to-peer dynamic, not salesperson-to-prospect.

**Template:**
```
no worries if the timings off — just didnt want you to miss it. lmk if you want me to send over the details
```

**Rules:**
- All lowercase
- "timings" not "timing's" (intentional)
- "didnt" not "didn't" (intentional)
- "lmk" is deliberate — casual, peer-level
- This is the LAST unsolicited text. If no reply after Text 3, the lead goes dormant.
- No 4th text. No "just checking in." No follow-up drip. Silence is doctrine.

---

## The 7 Human Texture Rules (Applied to All 3 Texts)

1. **All lowercase.** Capital letters signal corporate. Lowercase signals human.
2. **No apostrophes in contractions.** "dont" "cant" "wont" "youre" "its" — this is how people actually text.
3. **No periods at end of messages.** Periods in texts read as passive-aggressive or formal.
4. **Abbreviations allowed.** "lmk" "min" "msg" — but only ones a 30-40 year old professional would use.
5. **One thought per message.** No compound sentences. No bullet points. No walls of text.
6. **No emojis in first 3 texts.** Emojis signal sales. The first 3 texts are pure legitimacy.
7. **Typo tolerance.** If a minor typo appears naturally, leave it. Perfection signals automation.

## Delivery Rules

- Text 1 sends immediately upon lead entering "New Lead" stage
- Text 2 sends 24 hours after Text 1 IF no reply
- Text 3 sends 48 hours after Text 2 IF no reply
- After Text 3 with no reply: lead moves to dormant. No further outreach for 30 days.
- If lead replies at ANY point: immediately route to Intent Declaration Agent (Agent #2)
- All texts pass through Human Texture utility before GHL SMS send
- All texts pass through DoctrineKernelEnforcer compliance check before send
